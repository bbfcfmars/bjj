# BJJ Video Blog — AI Context

This folder is the **source of truth** for agents and developers working on the BJJ agent-curated video blog.

- **current-state.md** — Overview, stack, philosophy, autonomy, next steps
- **decisions.md** — ADR log (date, decision, options, outcome)
- **conventions.md** — Naming, folders, testing
- **dependencies.md** — Major deps and rationale
- **deployment.md** — Hosting and env
- **known-issues.md** — Open issues and limitations
- **examples.md** — Sample data and prompts

Starter prompts and artifacts live in `_notes/`:
- `_notes/bjj-starter.md` — Architecture deck, folder structure, .cursorrules, process_video.py blueprint, README
- `_notes/bjj-editor-prompt.md` — Editor-agent export format (clip naming, /public/videos)
